# letters-and-numbers
A solver for the numbers portion of the ABC show, Letters and Numbers.

## Usage

```
$ python driver.py <numbers...> <target>
```
For example:
```
$ python driver.py 7 7 9 8 25 75 711
75 + 25 + 9 + 8 + 7 + 7 = 131
75 + 7 + 7 + 25 + 8 * 9 = 186
75 + 7 + 7 + 8 + 25 * 9 = 322
25 + 7 + 7 + 8 + 75 * 9 = 722
25 + 7 + 7 + 8 * (75 + 9) = 711
```
